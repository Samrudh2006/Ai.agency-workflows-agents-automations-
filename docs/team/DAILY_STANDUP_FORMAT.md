# 🌅 Daily Standup Format
**Keeping the team aligned for 15 minutes a day.**

</div>

<br />

We operate an asynchronous daily standup in Slack to save time, followed by an optional 15-minute voice huddle specifically to unblock team members.

---

## 📝 The Asynchronous Standup (Posted by 10 AM Local Time)

Every morning, team members must post their update in the `#daily-standup` channel using the exact format below.

```markdown
**Yesterday:**
- Finished building the Make.com webhook for the Real Estate client.
- Wrote the knowledge base doc for the Support Chatbot.

**Today:**
- QA testing the Real Estate voice agent. Focus on interrupt handling.
- Having a discovery call with a new Dental client at 2PM.

**Blockers:**
- None. / YES: I am waiting on the client to send us their Twilio sub-account credentials so I can't finalize the phone routing.
```

## 🎙️ The 15-Minute Sync (Optional)

At 10:15 AM, the Lead Dev starts a Slack Huddle.
- **Goal:** Resolve the **Blockers** mentioned in the async updates.
- **Rule:** This is NOT a status update meeting. We already read what you are doing. This meeting is solely for asking questions like: *"You said you were blocked on the Make.com parsing error, screen-share real quick and I'll show you how to map the JSON array."*
- If there are no blockers, the huddle ends in 2 minutes.
